🧩 Task 1: Basic Output Control

Design a simple system that controls an output device such as an LED indicator or warning light. Think about where such systems are used in real life.

  • Example: LED control system
  • Example: Indicator light
  • Example: Warning signal system

🧩 Task 2: Variable Control System

Design a system where the output behavior changes gradually or dynamically.

  • Brightness control
  • Speed control
  • Intensity variation
  • Real-life use: Smart lighting or automation

🧩 Task 3: User Interaction System

Design a system that responds to user input and interaction.

  • Button-based control
  • Switch-based logic
  • Manual override systems
  • Human-machine interaction

🧩 Task 4: Alert or Notification System

Design a system that provides sound or alert-based feedback.

  • Buzzer alerts
  • Warning sounds
  • Notification signals
  • Safety systems or alarms

🧩 Task 5: Display or Information System

Design a system that displays information to the user.

  • Text display
  • Sensor value display
  • System status indicators
  • Monitoring dashboards

🧩 Task 6: Environmental Sensing System

Design a system that senses environmental conditions.

  • Light detection
  • Object detection
  • Environmental awareness
  • Smart city or energy-saving systems

🧩 Task 7: Monitoring System

Design a system that monitors physical or environmental parameters.

  • Temperature monitoring
  • Humidity monitoring
  • Comfort monitoring
  • Applications: smart homes, agriculture, health

🧩 Task 8: Motion or Actuation System

Design a system involving movement or mechanical action.

  • Motor control
  • Angle control
  • Positioning system
  • Applications: robotics and automation
