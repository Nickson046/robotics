Task 7 – Distance Sensor

Objective

Measure distance using an ultrasonic sensor.

Circuit Design

Wokwi Simulation

Code

int trig = 9;
int echo = 10;

void setup(){
 pinMode(trig, OUTPUT);
 pinMode(echo, INPUT);
 Serial.begin(9600);
}

void loop(){

 digitalWrite(trig, LOW);
 delayMicroseconds(2);

 digitalWrite(trig, HIGH);
 delayMicroseconds(10);
 digitalWrite(trig, LOW);

 long duration = pulseIn(echo, HIGH);
 int distance = duration * 0.034 / 2;

 Serial.println(distance);

 delay(500);

}

Explanation

The ultrasonic sensor sends sound waves and measures the time taken for the echo to return to calculate distance.
