Task 07 · Monitoring System

Smart Patient
Monitoring System

Monitors heart rate, SpO2, body temperature and room environment — triggers alarms when vitals go critical.

Objective

Build a hospital-grade patient monitoring system that tracks multiple vital signs simultaneously. Three potentiometers simulate heart rate, blood oxygen (SpO2), and body temperature. Two DHT22 sensors monitor patient and room environment. The system provides status LEDs, categorised alarms, a rotating 4-screen LCD, and a serial monitor vitals dashboard.

Components Used

  • Arduino Uno
  • 2 × DHT22 Temperature + Humidity Sensors
  • 3 × Potentiometers (heart rate / SpO2 / body temp)
  • 16×2 LCD with I2C
  • Piezo Buzzer
  • Green, Yellow, Red, Blue LEDs
  • Reset Push Button
  • 4 × 220Ω + 2 × 10kΩ Resistors
  • Breadboard + Jumper Wires

Vital Sign Ranges

  • Heart rate: Normal 60–100 bpm / Warning 50–59 or 101–120 / Critical <50 or >120
  • SpO2: Normal 95–100% / Warning 90–94% / Critical <90%
  • Body temp: Normal 36–37.5°C / Warning 37.5–39°C / Critical >39°C or <35°C

LCD Screens (rotate every 4 seconds)

  • Screen 1 — Heart rate + SpO2 with status
  • Screen 2 — Body temperature + overall status
  • Screen 3 — Room temperature + humidity
  • Screen 4 — All vitals summary at a glance

Code (key loop logic)

void loop() {
  unsigned long now = millis();

  // Read vitals from potentiometers
  bpm   = map(analogRead(POT_HR),   0, 1023, 30,  170);
  spo2  = map(analogRead(POT_SPO2), 0, 1023, 78,  100);
  btemp = map(analogRead(POT_TEMP), 0, 1023, 330, 420) / 10.0;

  // Read room DHT22
  float h = dhtRoom.readHumidity();
  float t = dhtRoom.readTemperature();
  if (!isnan(h)) rhumid = h;
  if (!isnan(t)) rtemp  = t;

  // Check for critical and trigger emergency lockout
  if (overall() == 2 && !alarmActive) {
    alarmActive = true;
    Serial.println("!!! CRITICAL ALERT TRIGGERED !!!");
  }

  // Reset button clears alarm
  if (digitalRead(RESET_BTN) == LOW && alarmActive) {
    alarmActive = false;
    noTone(BUZZER);
  }

  // Update LEDs, buzzer, LCD, serial
  updateLEDs();
  updateBuzzer(now);
  if (now - lastScreen >= 3000) {
    lcd.clear();
    screen = (screen + 1) % 4;
    lastScreen = now;
  }
  switch (screen) {
    case 0: showScreen0(); break;
    case 1: showScreen1(); break;
    case 2: showScreen2(); break;
    case 3: showScreen3(); break;
  }
  delay(200);
}

How to Test

Scroll POT1 down to simulate bradycardia (30 bpm) — red LED lights and rapid alarm triggers. Scroll POT2 down for hypoxia (80% SpO2) — blue LED activates. Scroll POT3 up for high fever (41°C). Once a critical alarm locks, press the green reset button to acknowledge and clear it. The Serial Monitor prints a full vitals table every 2 seconds.
