Task 1 – Basic Output Control

Objective

Create a simple LED control circuit.

Circuit Design

Wokwi Simulation

Code

int led = 13;

void setup(){
 pinMode(led, OUTPUT);
}

void loop(){
 digitalWrite(led, HIGH);
 delay(1000);
 digitalWrite(led, LOW);
 delay(1000);
}

Explanation

This circuit demonstrates basic digital output control.
