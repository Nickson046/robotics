Task 6 – Servo Motor Control

Objective

Control the position of a servo motor using Arduino.

Circuit Design

Wokwi Simulation

Code

#include 

Servo myServo;

void setup(){
 myServo.attach(9);
}

void loop(){

 myServo.write(0);
 delay(1000);

 myServo.write(90);
 delay(1000);

 myServo.write(180);
 delay(1000);

}

Explanation

The servo motor rotates to specific angles based on commands sent from the Arduino.
