Task 4 – Sensor Integration

Objective

Use a light sensor (LDR) to control an LED automatically.

Circuit Design

Wokwi Simulation

Code

int ldr = A0;
int led = 13;

void setup(){
 pinMode(led, OUTPUT);
}

void loop(){

 int light = analogRead(ldr);

 if(light < 500){
   digitalWrite(led, HIGH);
 }
 else{
   digitalWrite(led, LOW);
 }

}

Explanation

The LDR measures ambient light levels. When it becomes dark, the LED automatically turns on.
