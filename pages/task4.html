Task 04 · Alert System

Car Reverse
Parking Alert

Ultrasonic sensor + 5 LEDs + buzzer + LCD display — warns the driver as they reverse toward an obstacle.

Objective

Build a car reverse parking assistant that uses an HC-SR04 ultrasonic sensor to detect obstacles, displaying distance on an LCD, showing proximity with a 5-LED bar graph, and sounding increasingly urgent buzzer beeps. Simulates the parking assist systems used in modern vehicles.

Components Used

  • Arduino Uno
  • HC-SR04 Ultrasonic Sensor
  • 16×2 LCD with I2C (address 0x27)
  • Piezo Buzzer
  • 2 × Green LEDs, 1 × Yellow, 1 × Orange, 1 × Red
  • 5 × 220Ω Resistors
  • Breadboard + Jumper Wires

Distance Zones

  • > 80cm — SAFE: no LEDs, silent
  • 50–80cm — FAR: 1 green LED, slow beep
  • 30–50cm — MEDIUM: 2 green LEDs, medium beep
  • 15–30cm — CLOSE: 3 LEDs, fast beep
  • 5–15cm — DANGER: 4 LEDs, rapid beep
  • < 5cm — STOP NOW: all 5 LEDs, solid alarm tone

Wokwi Simulation

Code

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);
const int TRIG_PIN = 2, ECHO_PIN = 3;
const int LED_PINS[5] = {4, 5, 6, 7, 8};
const int BUZZER_PIN  = 9;

long getDistance() {
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  long dur = pulseIn(ECHO_PIN, HIGH, 30000);
  if (dur == 0) return 999;
  return dur * 0.034 / 2;
}

void loop() {
  long dist = getDistance();
  int zone;
  if      (dist > 80) zone = 0;
  else if (dist > 50) zone = 1;
  else if (dist > 30) zone = 2;
  else if (dist > 15) zone = 3;
  else if (dist > 5)  zone = 4;
  else                zone = 5;

  // Light LEDs equal to zone
  for (int i = 0; i < 5; i++) {
    digitalWrite(LED_PINS[i], i < zone ? HIGH : LOW);
  }

  // Buzzer: beep speed increases with zone
  if (zone == 0)      noTone(BUZZER_PIN);
  else if (zone == 5) tone(BUZZER_PIN, 1000);
  else {
    int intervals[6] = {0, 800, 500, 250, 100, 0};
    // beep logic handled with millis()
  }

  // Update LCD on zone change
  if (zone != lastZone) {
    updateLCD(dist, zone);
    lastZone = zone;
  }
  delay(100);
}

How It Works

The HC-SR04 fires a 10μs ultrasonic pulse and measures the echo return time. Distance = (duration × 0.034) / 2. The result is mapped to one of 6 distance zones. Each zone lights a corresponding number of LEDs and sets a beep interval — faster as the distance decreases. The LCD updates only on zone change to avoid flicker. In Wokwi, drag the sensor slider left to simulate reversing toward an obstacle.
