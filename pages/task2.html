Task 02 · Variable Control System

Smart Street Light
Brightness Control

Street lights automatically adjust brightness based on ambient light detected by an LDR sensor.

Objective

Create a smart street lighting system that automatically adjusts LED brightness based on the environment's light level. A potentiometer allows threshold adjustment, and an override button forces lights on regardless of the sensor reading — simulating real-world smart city lighting.

Components Used

  • Arduino Uno
  • Photoresistor (LDR)
  • Potentiometer — threshold adjustment
  • Push button — manual override
  • 6 × Yellow LEDs (street lights)
  • 6 × 220Ω Resistors + 10kΩ pull-down
  • Breadboard + Jumper Wires

Brightness Modes

  • Bright environment → all LEDs OFF (energy saving)
  • Twilight zone → LEDs at 30% brightness (DIM mode)
  • Full darkness → LEDs at 100% brightness (FULL mode)
  • Override button pressed → forced full brightness

Code

const int LDR_PIN  = A0;
const int POT_PIN  = A1;
const int BTN_PIN  = 2;
const int LED_PINS[6] = {3, 5, 6, 9, 10, 11};

const int BRIGHTNESS_OFF  = 0;
const int BRIGHTNESS_DIM  = 70;
const int BRIGHTNESS_FULL = 255;

int  currentMode    = -1;
bool overrideActive = false;
unsigned long lastPress = 0;

void fadeTo(int target) {
  bool goingUp = target > currentBrightness[0];
  while (currentBrightness[0] != target) {
    for (int i = 0; i < 6; i++) {
      currentBrightness[i] += goingUp ? 3 : -3;
      currentBrightness[i] = constrain(currentBrightness[i], 0, 255);
      analogWrite(LED_PINS[i], currentBrightness[i]);
    }
    delay(8);
  }
}

void loop() {
  int rawLDR  = analogRead(LDR_PIN);
  int potVal  = analogRead(POT_PIN);
  int ldrLight = 1023 - rawLDR;
  int threshold = map(potVal, 0, 1023, 200, 800);

  if (digitalRead(BTN_PIN) == LOW && millis() - lastPress > 300) {
    overrideActive = !overrideActive;
    lastPress = millis();
  }

  int newMode;
  if (overrideActive)             newMode = 3;
  else if (ldrLight > threshold + 60) newMode = 0;
  else if (ldrLight > threshold - 60) newMode = 1;
  else                                newMode = 2;

  if (newMode != currentMode) {
    // fade to appropriate brightness
    currentMode = newMode;
  }
  delay(400);
}

How It Works

The LDR forms a voltage divider — as it gets darker, the analog reading drops. The Arduino compares this to the threshold set by the potentiometer. A 120-unit hysteresis band prevents flickering at the threshold boundary. The override button toggles forced-on mode, simulating a manual switch found on real street light poles.
