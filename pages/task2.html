Task 02 · Variable Control System

Car Dashboard
Speedometer

A potentiometer acts as an accelerator pedal — speed increases gradually with auto gear shifting, LED bar graph, speed limit warnings, and a live ECU-style dashboard.

Objective

Design a variable control system where the output changes gradually and dynamically based on an analog input. A potentiometer simulates an accelerator pedal, controlling a virtual speed from 0 to 200 km/h. The system responds with automatic gear changes, a progressive LED bar graph, speed limit alerts, and a real-time LCD dashboard — mirroring how a real car instrument cluster behaves.

Real-Life Application

Every modern vehicle has an electronic control unit (ECU) that reads the accelerator pedal position and controls engine output, gear selection, instrument displays, and warning lights. This project simulates exactly that — the pot is the pedal, the Arduino is the ECU, and the LEDs and LCD are the instrument cluster. The speed limit warning system mirrors the over-speed alerts found in modern cars and commercial vehicle fleet management systems.

Components Used

  • Arduino Uno
  • Potentiometer 1 — accelerator pedal (A0)
  • Potentiometer 2 — adjustable speed limit (A1)
  • 16×2 LCD Display with I2C module
  • 2 × Green LEDs — low speed indicator
  • 2 × Yellow LEDs — medium speed indicator
  • 2 × Red LEDs — high speed / warning indicator
  • 6 × 220Ω Resistors
  • Piezo Buzzer — speed limit alarm
  • Breadboard + Jumper Wires

Speed Zones

  • 0–10 km/h → Gear N · 0 LEDs · Status: IDLE
  • 10–30 km/h → Gear 1 · 1 green LED · Status: CRAWL
  • 30–60 km/h → Gear 2 · 2 green LEDs · Status: CITY
  • 60–90 km/h → Gear 3 · 3 LEDs · Status: ROAD
  • 90–120 km/h → Gear 4 · 4 LEDs · Status: FAST
  • 120–150 km/h → Gear 5 · 5 LEDs · Status: HIGHWAY
  • 150+ km/h → Gear 6 · all LEDs flash · Status: DANGER

Code

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);

const int ACCEL_PIN   = A0;   // accelerator pot
const int LIMIT_PIN   = A1;   // speed limit pot
const int LED_PINS[5] = {3, 4, 5, 6, 7};
const int BUZZER_PIN  = 8;
const int WARN_LED    = 9;

int smoothedSpeed = 0;

int getGear(int speed) {
  if (speed < 10)  return 0;  // Neutral
  if (speed < 30)  return 1;
  if (speed < 55)  return 2;
  if (speed < 85)  return 3;
  if (speed < 115) return 4;
  if (speed < 150) return 5;
  return 6;
}

void loop() {
  int accelRaw = analogRead(ACCEL_PIN);
  int limitRaw = analogRead(LIMIT_PIN);

  // Smooth acceleration — can't jump instantly
  int targetSpeed = map(accelRaw, 0, 1023, 0, 200);
  if (targetSpeed > smoothedSpeed)
    smoothedSpeed = min(smoothedSpeed + 3, targetSpeed);
  else
    smoothedSpeed = max(smoothedSpeed - 4, targetSpeed);

  // Map pot2 to road speed limits
  int limitOptions[9] = {20,30,40,50,60,70,80,100,120};
  int limitIndex = map(limitRaw, 0, 1023, 0, 8);
  int speedLimit = limitOptions[limitIndex];

  bool limitBreached = (smoothedSpeed > speedLimit);

  // Update LED bar graph
  int LED_THRESH[5] = {20, 50, 80, 110, 140};
  for (int i = 0; i < 5; i++) {
    digitalWrite(LED_PINS[i], smoothedSpeed > LED_THRESH[i] ? HIGH : LOW);
  }

  // Warning LED flashes when over limit
  if (limitBreached) {
    digitalWrite(WARN_LED, (millis()/300) % 2 == 0 ? HIGH : LOW);
    tone(BUZZER_PIN, 1000);
  } else {
    digitalWrite(WARN_LED, LOW);
    noTone(BUZZER_PIN);
  }

  // LCD display
  lcd.setCursor(0, 0);
  lcd.print("SPD:");
  lcd.print(smoothedSpeed);
  lcd.print("km/h G:");
  lcd.print(getGear(smoothedSpeed));

  lcd.setCursor(0, 1);
  lcd.print("LIM:");
  lcd.print(speedLimit);
  lcd.print(limitBreached ? " OVER!" : " OK   ");

  delay(30);
}

How It Works

The accelerator potentiometer feeds an analog value (0–1023) into the Arduino which maps it to a speed of 0–200 km/h. Rather than jumping instantly to the target speed, a smoothing algorithm increases or decreases the displayed speed by a few km/h each loop — this simulates the inertia of a real vehicle accelerating and braking. The second potentiometer maps to one of nine real UK road speed limits (20, 30, 40, 50, 60, 70, 80, 100, 120 km/h). As speed crosses each LED threshold, the bar graph lights up progressively. When speed exceeds the limit, the warning LED flashes, the buzzer beeps with increasing urgency, and the LCD corner shows the breach. At over 160 km/h all LEDs flash rapidly in a danger pattern. A random speed camera event also fires occasionally to simulate real road monitoring.
