Task 2 – Variable Control System

Objective

Control LED brightness using a potentiometer.

Circuit Design

Wokwi Simulation

Code

int pot = A0;
int led = 9;

void setup(){
 pinMode(led, OUTPUT);
}

void loop(){

 int value = analogRead(pot);
 int brightness = map(value,0,1023,0,255);

 analogWrite(led, brightness);

}

Explanation

The potentiometer provides an analog input. The Arduino reads the value and adjusts LED brightness using PWM.
