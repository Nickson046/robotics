Task 3 – User Interaction

Objective

Create a system where a push button controls an LED.

Circuit Design

Wokwi Simulation

Code

int button = 2;
int led = 13;

void setup(){
 pinMode(button, INPUT);
 pinMode(led, OUTPUT);
}

void loop(){

 if(digitalRead(button) == HIGH){
   digitalWrite(led, HIGH);
 }
 else{
   digitalWrite(led, LOW);
 }

}

Explanation

The button acts as a digital input. When pressed, the Arduino receives a HIGH signal and turns on the LED.
