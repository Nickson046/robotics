Task 03 · User Interaction

Smart Room
Control Panel

3 buttons control light, fan, and alarm — with toggle, hold, and double-tap gestures.

Objective

Build a smart room control panel using push buttons to control three room devices — light, fan, and alarm. The system uses button gestures (tap, hold, double-tap) to unlock different functions, simulating a real home automation wall panel.

Components Used

  • Arduino Uno
  • 3 × Push Buttons (blue, green, red)
  • 3 × LEDs (yellow, green, red)
  • 3 × 220Ω Resistors
  • Breadboard + Jumper Wires

Button Functions

  • Blue button — tap to toggle light on/off
  • Green button — tap to toggle fan / hold 2s to cycle fan speed (low → med → high)
  • Red button — tap to toggle alarm / double-tap triggers 6-flash emergency sequence

Wokwi Simulation

Code

const int BTN_LIGHT = 2, BTN_FAN = 3, BTN_ALARM = 4;
const int LED_LIGHT = 9, LED_FAN = 10, LED_ALARM = 11;

bool lightOn = false, fanOn = false, alarmOn = false;
int  fanSpeed = 0;
const int FAN_SPEEDS[4] = {0, 80, 160, 255};

void loop() {
  // Light: simple toggle
  if (digitalRead(BTN_LIGHT) == LOW) {
    lightOn = !lightOn;
    analogWrite(LED_LIGHT, lightOn ? 255 : 0);
    delay(40);
  }

  // Fan: tap = toggle, hold 2s = cycle speed
  if (digitalRead(BTN_FAN) == LOW) {
    unsigned long held = 0;
    while (digitalRead(BTN_FAN) == LOW) {
      held++;
      delay(1);
    }
    if (held >= 2000) {
      fanSpeed = (fanSpeed + 1) % 4;
      fanOn = (fanSpeed != 0);
    } else {
      fanOn = !fanOn;
      if (!fanOn) fanSpeed = 0;
      else if (fanSpeed == 0) fanSpeed = 2;
    }
    analogWrite(LED_FAN, FAN_SPEEDS[fanSpeed]);
  }

  // Alarm: single = toggle, double-tap = flash
  if (digitalRead(BTN_ALARM) == LOW) {
    unsigned long gap = millis() - lastAlarmTap;
    if (gap < 400) {
      // Double tap - flash all LEDs
      for (int i = 0; i < 6; i++) {
        analogWrite(LED_LIGHT, 255);
        analogWrite(LED_FAN, 255);
        analogWrite(LED_ALARM, 255);
        delay(120);
        analogWrite(LED_LIGHT, 0);
        analogWrite(LED_FAN, 0);
        analogWrite(LED_ALARM, 0);
        delay(120);
      }
    } else {
      alarmOn = !alarmOn;
      analogWrite(LED_ALARM, alarmOn ? 255 : 0);
    }
    lastAlarmTap = millis();
  }
  delay(20);
}

How It Works

Each button uses INPUT_PULLUP — reading LOW when pressed. The fan button measures how long it is held to decide between a quick tap and a hold gesture. The alarm button tracks the time since the last press to detect a double-tap within 400ms. All LEDs use PWM-capable pins so fan speed can be shown as actual brightness dimming.
