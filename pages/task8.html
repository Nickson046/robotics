Task 8 – Final Integrated System

Objective

Combine sensors and outputs to build a simple automated system.

Circuit Design

Wokwi Simulation

Code

int sensor = A0;
int led = 13;

void setup(){
 pinMode(led, OUTPUT);
}

void loop(){

 int value = analogRead(sensor);

 if(value > 600){
   digitalWrite(led, HIGH);
 }
 else{
   digitalWrite(led, LOW);
 }

}

Explanation

This final task combines sensor input with automated output control.
