Task 08 · Motion System — Final Project

Automatic Door
System

PIR + ultrasonic sensors, dual servos, 3 operating modes, visitor counter, intruder alert, and LCD event log.

Objective

Build a fully-featured automatic door system as the final and most complex project of the semester. The system combines proximity sensing, motion detection, dual servo control, and three operating modes to simulate the automatic doors used in shopping malls, hospitals, and airports.

Components Used

  • Arduino Uno
  • HC-SR04 Ultrasonic Sensor (proximity zones)
  • PIR Motion Sensor (standby wake)
  • 2 × SG90 Servo Motors (main door + inner gate)
  • Piezo Buzzer (entry chime + alerts)
  • 16×2 LCD with I2C
  • Blue, Yellow, Green, Red LEDs (status indicators)
  • Mode button + Manual open button
  • 4 × 220Ω Resistors
  • Breadboard + Jumper Wires

Distance Zones

  • > 150cm — Standby: blue LED, PIR watching for motion
  • 80–150cm — Approaching: yellow LED, system wakes
  • 20–80cm — Open zone: both servos open, green LED, entry chime
  • < 20cm — Too close: red flash, warning beep

Operating Modes

  • AUTO — door opens and closes automatically based on sensor readings
  • MANUAL — door only opens when the manual button is pressed
  • LOCKED — door refuses to open; manual button plays rejected beep

Advanced Features

  • Visitor counter — increments each time someone passes through
  • Intruder alert — triggers if door is blocked open for more than 8 seconds
  • Startup self-test — servos sweep and buzzer chimes on boot
  • LCD 3-page rotation — door status / visitor count / event log
  • Serial event log with millisecond timestamps

Code (state machine core)

void loop() {
  long dist = getDistance();
  bool pirMotion = digitalRead(PIR_PIN);

  // Mode cycling via button
  if (modeButtonPressed()) {
    mode = (mode + 1) % 3;
    if (mode == MODE_LOCKED) { closeDoor(); closeGate(); }
  }

  if (mode == MODE_LOCKED) {
    setLED(LED_RED);
    playLockedBeep(now);
    return;
  }

  // Wake system on PIR motion
  if (pirMotion && !systemAwake) {
    systemAwake = true;
    logEvent("PIR motion detected");
  }

  // Intruder check: door open too long
  if (doorOpen && now - doorOpenTime > 8000) {
    intruderAlert = true;
    logEvent("INTRUDER ALERT");
  }

  // AUTO mode distance zones
  if (mode == MODE_AUTO) {
    if (dist < ZONE_OPEN) {
      setLED(LED_GREEN);
      openDoor(); openGate();
      if (!personCounted) {
        visitorCount++;
        personCounted = true;
      }
    } else if (dist < ZONE_APPROACH) {
      setLED(LED_YELLOW);
      if (doorOpen) { closeDoor(); closeGate(); }
    } else {
      setLED(LED_BLUE);
      if (doorOpen && now - doorOpenTime > 3000) {
        closeDoor(); closeGate();
      }
    }
  }
}

How to Test

On startup both servos do a self-test sweep and 3 ascending beeps play. Drag the HC-SR04 slider slowly left — at 80cm the yellow LED activates (approaching), at 30cm both servos open with a chime (green LED), visitor count increments. Leave the slider close for 8 seconds to trigger the intruder alarm. Press the blue mode button to cycle AUTO → MANUAL → LOCKED and observe how the system behaves differently in each mode.
