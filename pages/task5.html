Task 05 · Display System

Smart Energy
Monitor

Dual potentiometers simulate current and voltage — LCD shows watts, cost/hr, peak load, and status.

Objective

Design a smart electricity meter display that calculates and shows real-time power consumption. Two potentiometers simulate a current sensor and voltage fluctuation. The system displays power in watts, estimated cost per hour, peak readings, and triggers alarms at critical consumption levels.

Components Used

  • Arduino Uno
  • 2 × Potentiometers (current + voltage simulation)
  • 16×2 LCD with I2C
  • Piezo Buzzer
  • 2 × Green LEDs, 1 × Yellow, 1 × Orange, 1 × Red
  • 5 × 220Ω Resistors
  • Breadboard + Jumper Wires

Consumption Zones

  • 0–1000W — NORMAL: green LEDs, silent
  • 1000–2000W — HIGH: yellow LED, warning beep
  • 2000–3500W — VERY HIGH: orange LED, slow alarm
  • 3500W+ — CRITICAL: red LED, solid alarm, LCD alert

Wokwi Simulation

Code

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);

const int POT_CURRENT = A0, POT_VOLTAGE = A1;
const int LED_PINS[5] = {3, 4, 5, 6, 7};
const int BUZZER_PIN  = 8;

const float COST_PER_KWH = 0.15;
const float MAX_POWER    = 4800.0;

float peakWatts = 0;

void loop() {
  int rawI = analogRead(POT_CURRENT);
  int rawV = analogRead(POT_VOLTAGE);

  float current = map(rawI, 0, 1023, 0, 2000) / 100.0;
  float voltage = map(rawV, 0, 1023, 2100, 2400) / 10.0;
  float watts   = current * voltage;
  float cost    = (watts / 1000.0) * COST_PER_KWH;

  if (watts > peakWatts) peakWatts = watts;

  int ledsOn = constrain((int)((watts / MAX_POWER) * 5.0), 0, 5);
  for (int i = 0; i < 5; i++) {
    digitalWrite(LED_PINS[i], i < ledsOn ? HIGH : LOW);
  }

  // Update LCD every 500ms
  lcd.setCursor(0, 0);
  lcd.print((int)watts);
  lcd.print("W $");
  lcd.print(cost, 2);
  lcd.print("/h    ");

  lcd.setCursor(0, 1);
  if (watts > 3500) lcd.print("CRITICAL!       ");
  else if (watts > 2000) lcd.print("VERY HIGH       ");
  else if (watts > 1000) lcd.print("HIGH            ");
  else lcd.print("NORMAL          ");

  delay(500);
}

How It Works

POT1 maps to 0–20A (current) and POT2 maps to 210–240V. Power (watts) = current × voltage. Cost per hour = (watts ÷ 1000) × £0.15. The LCD alternates every 3 seconds between showing the power/cost view and a voltage/current/total-kWh view. The serial monitor prints a full formatted dashboard table every 2 seconds.
