Task 5 – Motor Control

Objective

Control the speed of a DC motor using PWM.

Circuit Design

Wokwi Simulation

Code

int led1 = 9;
int led2 = 10;
int led3 = 11;
int button = 2;

int speedLevel = 0;
bool lastState = HIGH;

void setup() {
  pinMode(led1, OUTPUT);
  pinMode(led2, OUTPUT);
  pinMode(led3, OUTPUT);
  pinMode(button, INPUT_PULLUP);
}

void loop() {
  bool currentState = digitalRead(button);

  if (lastState == HIGH && currentState == LOW) {
    speedLevel++;
    if (speedLevel > 3) speedLevel = 0;
    delay(200);
  }

  lastState = currentState;

  digitalWrite(led1, LOW);
  digitalWrite(led2, LOW);
  digitalWrite(led3, LOW);

  if (speedLevel == 1) digitalWrite(led1, HIGH);

  if (speedLevel == 2) {
    digitalWrite(led1, HIGH);
    digitalWrite(led2, HIGH);
  }

  if (speedLevel == 3) {
    digitalWrite(led1, HIGH);
    digitalWrite(led2, HIGH);
    digitalWrite(led3, HIGH);
  }
}

Explanation

In this task, a DC motor is simulated using three LEDs in the Wokwi environment. Since a real DC motor is not available, LEDs are used to visually represent different motor speed levels.

A pushbutton connected to pin 2 is used to change the speed level. Each time the button is pressed, the system cycles through four states: OFF, slow, medium, and high speed.

Instead of using actual motor rotation, the LEDs indicate speed as follows:

  • 0 LEDs ON → Motor OFF
  • 1 LED ON → Low speed
  • 2 LEDs ON → Medium speed
  • 3 LEDs ON → High speed

This demonstrates how motor speed can be controlled using different output levels. In real applications, this would be done using PWM signals with analogWrite(), but in this simulation, multiple LEDs are used to clearly visualize the changes.
